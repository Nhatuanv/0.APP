\# Coding Directives

\- \*\*Conciseness:\*\* Only output modified code blocks. Do not explain the code unless explicitly asked. 

\- \*\*Refactoring:\*\* Do not refactor existing code outside the scope of the requested feature.

\- \*\*Logging:\*\* Remove all `console.log` statements before finalizing a task.

\- \*\*Dependencies:\*\* Do not install new npm packages or Rust crates without asking for confirmation first.
